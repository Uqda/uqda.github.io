# شبكة عُقدة — الملخص التنفيذي

**نظرة عامة في صفحة واحدة**

---

## ما هي عُقدة؟

**شبكة عُقدة** هي بروتوكول شبكة mesh لامركزية مع تشفير من طرف إلى طرف وإعدادات دنيا. مبني على خوارزميات توجيه مثبتة، تمكن عُقدة من الاتصال المباشر بين الأقران عبر طوبولوجيات شبكية ديناميكية.

**الميزة الرئيسية**: تطور محسّن للأداء لشبكة Yggdrasil v0.5، يركز على النشر العملي والتميز التشغيلي.

---

## الميزات الرئيسية

| الميزة | الوصف |
|--------|-------------|
| **التشفير من طرف إلى طرف** | جميع حركة المرور مشفرة افتراضياً (ChaCha20-Poly1305) |
| **متوافق مع البروتوكول** | متوافق 100% مع عقد Yggdrasil v0.5 |
| **محسّن للأداء** | تحسين زمن الاستجابة بمقدار 20-50ms عن الأساسي |
| **شبكة Mesh ذاتية الإصلاح** | اكتشاف المسار واستعادته تلقائياً |
| **مستقل عن الموقع** | عنوان IPv6 دائم مشتق من الهوية |
| **إعدادات صفرية** | تتشكل الشبكات تلقائياً |
| **مجاني للأبد** | بدون تكلفة، بدون تسجيل، بدون سلطة مركزية |

---

## تحسينات الأداء

| المقياس | الأساسي | عُقدة | التحسين |
|--------|----------|------|-------------|
| إنشاء الاتصال | 50-100ms | 20-40ms | أسرع بنحو 60% |
| مهلة المصافحة | 6s | 5s | -16.7% |
| مهلة اتصال TCP | 5s | 3s | -40% |
| تأخير إعادة الاتصال الأولى | 1s | 100ms | -90% |
| زمن الاستجابة من طرف إلى طرف | الأساسي | -20-50ms | تحسين قابل للقياس |

**التأثير العملي**: في شبكة من 10 قفزات مع 3 أعطال مؤقتة في الساعة، تقلل هذه التحسينات متوسط زمن الاستجابة من طرف إلى طرف بمقدار **35-55ms**.

## البدء السريع

### التثبيت

**Linux:**
```bash
sudo dpkg -i uqda-debian-amd64.deb
```

**Windows:**
```powershell
msiexec /i uqda-windows-x64.msi
```

**macOS:**
```bash
# تحميل وفتح مثبت .pkg
```

### التشغيل

```bash
# الإعداد التلقائي (موصى به)
sudo uqda -autoconf

# مع ملف الإعدادات
uqda -genconf > uqda.conf
sudo uqda -useconffile uqda.conf
```

### المراقبة

```bash
# الحصول على عنوانك
uqdactl getSelf

# عرض الأقران
uqdactl getPeers

# إضافة نظير
uqdactl addPeer tcp://peer.example.com:12345
```

## المواصفات التقنية

- **إصدار البروتوكول**: متوافق مع 0.5 (مع Yggdrasil)
- **التشفير**: ChaCha20-Poly1305
- **التوقيع**: Ed25519
- **تبادل المفاتيح**: X25519
- **مساحة العناوين**: IPv6 `0200::/7`
- **النقل**: TCP/TLS/QUIC/WS/WSS
- **التوجيه**: Distance-Vector (DHT) + Spanning Tree

---

## حالة الأمان

- **الحالي**: لم يتم تدقيقه بشكل مستقل
- **يرث**: مكتبة Go القياسية (مدققة) + Yggdrasil (مراجعة مجتمعية)
- **مخطط**: تدقيق ممول من المجتمع Q2 2026
- **التوصية**: استخدم جدران الحماية المناسبة، لا تعرض الخدمات الحساسة
