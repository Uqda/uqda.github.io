// البدء

عُقدة مكتوبة بـ Go. جميع كود المصدر الخاص بنا متاح على GitHub. قاعدة الكود صغيرة نسبياً وسهلة التنقل.

نرحب بالمساهمات، إما في شكل Pull Requests أو Issues. نظراً لأن هذا مشروع هواة إلى حد كبير في هذه المرحلة، قد يستغرق الأمر بعض الوقت للوصول إلى مراجعة PRs/issues.

// المنصات

تم اختبار عُقدة على عدد من المنصات (بما في ذلك Linux، Windows، macOS، FreeBSD، OpenBSD). تتوفر أيضاً حزم لعدد من التوزيعات/المنصات.

// البناء من المصدر

المتطلبات الأساسية:

  • Go 1.22 أو أحدث
  • Git
git clone https://github.com/Uqda/Core.git
cd Core
./build
