// SERVICES ON UQDA NETWORK

The following services are available on the Uqda Network, courtesy of our community users. If you host a service on the Uqda Network or find that a service is no longer available, please feel free to open an issue on GitHub.

// ADDING YOUR SERVICE

To add your service to this list, please open an issue on GitHub with the following information:

  • Service name and description
  • Uqda IPv6 address and port (if applicable)
  • Access URL (if applicable)
  • Contact information (optional)
// COMMUNITY SERVICES

The Uqda Network community is growing, and services are being added regularly. Check back often for new services!

Note: Services listed here are provided by community members. Uqda Network does not endorse or guarantee the availability of any listed service.

// GET STARTED

To access services on the Uqda Network, you need to:

  1. Install Uqda on your system
  2. Configure your node and connect to peers
  3. Use the Uqda IPv6 addresses to access services
